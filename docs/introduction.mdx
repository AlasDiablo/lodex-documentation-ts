# Documentation

## Bienvenue sur la documentation du logiciel de visualisation open source Lodex !

Bienvenue sur la documentation du logiciel de visualisation open source Lodex !

Tout comme les données que vous visualiserez, Lodex évolue en permanence.
Cela signifie que parfois, le parcours peut être semé de défis. Cependant,
nous croyons fermement que chaque obstacle est une opportunité d’apprentissage.
Si vous avez des interrogations,
contactez-nous directement via le [formulaire de contact](https://www.lodex.fr/contact/).

**Pour plonger directement dans l’action, explorez le menu.
Vous y découvrirez une multitude d’astuces pour débuter en douceur,
que vous soyez développeur et que vous souhaitez installer le logiciel ou
simplement curieux d’explorer ses fonctionnalités clés.**

**Si vous préférez une approche plus détaillée,
vous trouverez ci-dessous un tutoriel interactif pas à pas.**
